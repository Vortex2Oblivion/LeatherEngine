interface IVideo

package hxvlc.openfl

import hxvlc.openfl.Video

extended by Video

Interface representing a video object with playback and media management functionalities.

Methods

dispose():Void

Frees the memory used by the media object.

getMeta(e_meta:Int):String

Retrieves metadata for the current media item.

Parameters:

e_meta

The metadata type.

Returns:

The metadata value as a string, or null if not available.

load(location:Location, ?options:Array<String>):Bool

Loads media from the specified location.

Parameters:

location

The location of the media file or stream.

options

Additional options to configure the media.

Returns:

true if the media was loaded successfully, false otherwise.

loadFromSubItem(index:Int, ?options:Array<String>):Bool

Loads a media subitem from the current media's subitems list at the specified index.

Parameters:

index

The index of the subitem to load.

options

Additional options to configure the loaded subitem.

Returns:

true if the subitem was loaded successfully, false otherwise.

nextChapter():Void

Moves to the next chapter, if supported.

parseStop():Void

Stops parsing the current media item.

parseWithOptions(parse_flag:Int, timeout:Int):Bool

Parses the current media item with the specified options.

Parameters:

parse_flag

The parsing option.

timeout

The timeout in milliseconds.

Returns:

true if parsing succeeded, false otherwise.

pause():Void

Pauses playback.

play():Bool

Starts playback.

Returns:

true if playback started successfully, false otherwise.

previousChapter():Void

Moves to the previous chapter, if supported.

resume():Void

Resumes playback.

saveMeta():Bool

Saves the metadata of the current media item.

Returns:

true if the metadata was saved successfully, false otherwise.

setMeta(e_meta:Int, value:String):Void

Sets metadata for the current media item.

Parameters:

e_meta

The metadata type.

value

The metadata value.

stop():Void

Stops playback.

togglePaused():Void

Toggles the pause state.
